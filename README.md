# Nexus Vision // Image Lab

A beautiful, dark-themed sovereign gateway for AI image generation. 

Built with:
- **Tailwind CSS**: Modern utility-first styling.
- **Pollinations.ai**: Backend for rapid, free visual synthesis.
- **Caddy**: Secure reverse proxy for key management and SSL.

## Features
- [x] Dynamic Model Discovery
- [x] Live Pollen Balance Tracking
- [x] Secure "Keyless" Frontend via Sovereign Proxy
- [x] Responsive Design for Mobile and Desktop

## Nexus Architecture
This lab is part of the **Nexus Sovereign Proxy**, allowing for private, encrypted AI workflows without public exposure.
